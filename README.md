# 📊 Personal Finance Analysis Dashboard

An interactive Excel dashboard analyzing **13 months of personal bank transaction data** (Apr 2025 – May 2026), built to uncover spending patterns, savings trends, and financial health insights.

---

## 🎯 Project Overview

This project transforms raw bank statement data into actionable financial insights through an 8-sheet interactive Excel workbook featuring KPIs, dynamic charts, and a custom Financial Health Score.

**Key Metrics Analyzed:**
- 💰 Total Income: ₹18,64,604
- 💸 Total Expense: ₹19,06,086
- 📉 Net Savings: ₹-41,481
- 🔢 Total Transactions: 908

---

## 🔧 Tools & Techniques Used

- **Microsoft Excel** (Advanced)
- **Formulas:** XLOOKUP, SUMIFS, IFERROR, TEXT, INDEX/MATCH
- **Pivot Tables & Charts**
- **Dynamic Visualizations:** Bar, Donut, Line, Scatter, Combo charts
- **Conditional Formatting**
- **Dashboard Design** (Dark Theme UI)

---

## 📂 Workbook Structure (9 Sheets)

| # | Sheet | Purpose |
|---|---|---|
| 1 | **Data** | Raw bank transaction data |
| 2 | **Dashboard** | Main KPI summary view |
| 3 | **Overview** | Financial Health Score + Timeline |
| 4 | **Income** | Salary vs Investment breakdown |
| 5 | **Expense** | Category-wise expense analysis |
| 6 | **Trends** | Month-wise income vs expense analysis |
| 7 | **BarChart** | Bar chart visualizations |
| 8 | **PieChart** | Pie chart visualizations |
| 9 | **LineChart** | Line chart visualizations |

## 💡 Key Insights Delivered

1. **Deficit Dominance:** 11 of 13 months ran a deficit
2. **UPI Expense Dominance:** 53.6% of all expenses are UPI-based
3. **EMI Burden:** EMI + Loan payments = 24.9% of outflows
4. **Stable Salary Base:** Monthly income between ₹1.09L–₹1.30L
5. **Savings Opportunity:** ~₹6L annual savings potential identified

## 📥 How to Use

1. Download the `.xlsx` file from this repository
2. Open in Microsoft Excel (2019 or later recommended)
3. Navigate using the side panel buttons
4. Use month filters at the top of the Dashboard
